# Інформаційний портал: Вітрова електростанція «Приморська»

Даний проєкт є практичною роботою №1 з дисципліни **«Основи веб-програмування»**. Це адаптивний односторінковий сайт енергетичного об'єкта.

## Загальна інформація
* **Об'єкт:** Вітрова електростанція «Приморська» (Варіант 3).
* **Мета:** Спроєктувати та реалізувати HTML-структуру і CSS-стилі з використанням Bootstrap 5.
* **Технології:** HTML5, CSS3, Bootstrap 5.3.

## Функціонал проєкту
1. **Навігаційне меню:** Адаптивний Navbar із плавною прокруткою до розділів.
2. **Героїчна секція:** Заголовок `<h1>` та підзаголовок з анімацією появи.
3. **Опис об'єкта:** Детальна інформація про історію та призначення ВЕС.
4. **Технічні параметри:** Таблиця з характеристиками об'єкта (потужність, напруга, обладнання).
5. **Медіа:** Адаптивні зображення та схеми обладнання.
6. **Адаптивність:** Коректне відображення на мобільних пристроях, планшетах та ПК.

##  Як запустити
1. Клонуйте або завантажте репозиторій.
2. Переконайтеся, що папка `images` містить необхідні файли.
3. Відкрийте файл `index.html` у будь-якому сучасному браузері.
